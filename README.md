![Kotlin](https://img.shields.io/badge/Kotlin-0095D5?logo=kotlin&logoColor=white)

Простое приложение для отслеживания ежедневных привычек с визуализацией прогресса по дням недели.

---

## Основные возможности

- **Два экрана навигации**:
  - Список привычек с карточками
  - Экран создания/редактирования привычки
- **Интерактивные карточки**:
  - Название привычки
  - Статус выполнения
  - Визуальный прогресс по дням недели
- **Управление выполнением**:
  - Отметка выполнения привычки одним касанием
  - Кликабельный сегодняшний день для быстрого обновления статуса
  - Автоматическое обновление прогресса при изменении состояния
- **Управление данными**:
  - Создание привычки с валидацией (запрет пустого названия)
  - Удаление привычек свайпом влево/вправо
  - Сохранение состояния при повороте экрана и смене конфигурации
- **Плавная навигация**:
  - Мгновенное отображение новой привычки после сохранения
  - Анимированные переходы между экранами

---

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| **Язык** | Kotlin 2.0.21 |
| **UI** | Jetpack Compose |
| **Архитектура** | MVVM + ViewModel + StateFlow |
| **Хранение данных** | Room Persistence Library |
| **Навигация** | Jetpack Navigation Compose |
| **DI** | KSP (Kotlin Symbol Processing) для генерации кода Room |
| **Сборка** | Gradle 8.9 + AGP 9.0.0 |
| **Min SDK** | 24 (Android 7.0) |
